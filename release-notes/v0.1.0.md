## Client
- Added client-background.png as a packaged UI resource at `/src/net/
  runelite/client/ui/client-background.png` and applied it to all RL interfaces.
- Deleted in the working tree: old grounditems, loottracker, npcnames, pathtracing, and tileindicators plugin files.
- Implemented window position/state persistence
- Added many missing `net.runelite.api.events` classes including chat, scripts, widgets, varbits, projectiles, graphics
  objects, object spawn/despawn/change, clan/friends chat, GE, player/NPC, sound, world, and composition events.
- Added multiple missing `net.runelite.client.events` classes, including chat input, config sync, external plugin
  changes, party events, loot events, plugin messages, profile changes, screenshots, and worlds fetch.
- Expanded `GameEventBridgeHooks.java` with
  bridge methods for the new events.
- Added `TileObjectInfo` payload support so this deob can emit object events even without native RuneLite `TileObject`
  wrappers.
- Added deferred event posting through `DeferredEventBus`.
- Updated `InputStream_Sub2.java:37` zone packet handling.
    - Mapped `LOC_ADD_CHANGE` to wall/decorative/game/ground object spawned events.
    - Mapped `LOC_DEL` to wall/decorative/game/ground object despawned events.
    - Mapped `MAP_ANIM` to `GraphicsObjectCreated`.
    - Mapped `MAP_PROJANIM` and `MAP_PROJANIM_HALFSQ` to `ProjectileMoved`.
    - Mapped `SOUND_AREA` and `SOUND_VORBIS_AREA` to `AreaSoundEffectPlayed`.
    - Existing ground item zone packets now route through `ItemSpawned`, `ItemDespawned`, and `ItemQuantityChanged`.
- Updated `Class348_Sub42_Sub8_Sub2.java`
- Mapped `RUNSCRIPT` to `ScriptPreFired`, `ScriptCallbackEvent`, and `ScriptPostFired`.
- Mapped public chat packet to `ChatMessage`.
- Mapped stat update packet to `StatChanged`.
- Mapped varbit packet to `VarbitChanged`.
- Mapped partial inventory update packet to `ItemContainerChanged`.
- Added `PostClientTick`.
- Added polled `CanvasSizeChanged`.
- Added polled `FocusChanged`.
- Reworked XP tracker 1:1 RuneLite’s original XP tracker layout.
- Moved the plugin sidebar open button into the titlebar area next to window controls.
- Added/updated titlebar layout logic so the sidebar button does not overlap minimize/maximize/close controls.

## Usage

Run with:

```bash
java -jar runelite-void-client-0.0.5.jar --address xxx.xxx.xxx.xxx --port xxxxx
```
